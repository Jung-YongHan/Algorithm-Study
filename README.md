## 📜규칙
### 스터디 방식
- `매일 1문제`
- 각자에 맞는 solved.ac 클래스 문제 선정
- 코드 파일과 해설 파일 업로드

### 문제 풀이
- `코드 파일`과 풀이를 설명한 `마크다운 파일` 본인 브렌치에 커밋 후 pr 생성

### PR 및 Commit Message 규칙
- `Pull Request`
  - [본인 이름] yy-mm-dd (ex. [김건우] 22-10-08)
- `Commit Message`
  - [문제 출처] 문제 이름 (ex. [BOJ-1920] 수 찾기)

### 파일 및 폴더 구조
